######  需求：

商场抽奖，一二三等奖，没有空奖，一等奖20名，二等奖30名，三等奖50名。

###### 思路：
生成数组[1,2,3....100]，随机挑选一个数字，如果数字在[1,20]区间，则为一等奖，[21,50]区间为二等奖，[51，100]区间为三等奖。

页面使用jQueryRotate进行转动实现。
